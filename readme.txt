== Description ==

OnBuy Integration for WooCommerce allows user to sync their WooCommerce store and its products to the OnBuy Marketplace. This plugin can also be used to manage the OnBuy orders as well directly from WooCommerce Store.

A few notes about the sections above:

*   Upload Simple Products to OnBuy Marketplace
*   Upload Variable Products to OnBuy Marketplace
*   Auto Sync Price and Inventory
*   Manage OnBuy Orders directly on WooCommerce

= Minimum Requirements =

* WordPress 4.7 or greater
* WooCommerce 3.0.0 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater
